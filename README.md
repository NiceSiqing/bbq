# BBQ Chicken - 韩式炸鸡订餐系统 🍗

一个现代化、响应式的韩式炸鸡订餐 Web 应用，基于 React、TypeScript 和 Tailwind CSS 构建。

**在线演示**：[https://siqing-bbq-chicken-project.netlify.app](https://siqing-bbq-chicken-project.netlify.app)

## 🛠️ 技术栈

### 前端框架

- **React 19.1** - 使用最新特性的现代 React
- **TypeScript 5.8** - 类型安全的开发体验
- **Tailwind CSS 4.1** - 实用优先的样式框架，支持自定义 rem 单位
- **React Router Dom 7.7** - 客户端路由管理

### 状态管理

- **Zustand 5.0** - 轻量级状态管理库
- **持久化中间件** - 本地存储集成，实现会话持久化

### 构建工具

- **Vite 7.0** - 快速的开发服务器和构建工具
- **ESLint** - 代码规范检查和质量保证
- **PostCSS** - CSS 后处理器

### 部署平台

- **Netlify** - 部署地址：[siqing-bbq-chicken-project.netlify.app](https://siqing-bbq-chicken-project.netlify.app)

## 📊 功能详解

### 订单模拟

- **真实时间模拟**：30–60 分钟预估配送时间，±5 分钟波动
- **状态流转**：待处理 → 制作中 → 配送中 → 已完成（含自动时间推进）
- **自动更新**：模拟真实商家接单流程和定时状态变更

### 购物车管理

- **智能更新**：合并相同商品，自动移除零数量商品
- **数据持久化**：页面刷新后在同一浏览器中保持购物车状态
- **实时计算**：即时价格计算和商品数量统计

### 用户体验

- **会话管理**：订单完成后自动清理会话数据
- **错误处理**：针对不完整会话的优雅导航
- **无障碍支持**：键盘导航和屏幕阅读器支持

## 📱 页面和组件

### 页面结构

- **首页** (`/`)：Hero 区域、品牌介绍、推荐产品
- **菜单页** (`/menu`)：展示所有炸鸡口味
- **姓名录入页** (`/name`)：输入用户名，开启点餐流程
- **点餐页** (`/order`)：添加商品至购物车
- **购物车页** (`/cart`)：修改、确认订单商品
- **结账页** (`/checkout`)：填写用户信息、下单
- **订单状态页** (`/OrderStatus`)：实时配送状态更新

### 核心组件

- `Header` - 顶部导航栏
- `Footer` - 页脚信息
- `Carousel` - 炸鸡口味轮播图
- `MenuCard` - 商品卡片展示
- `OrderOnline` - 首页下单按钮
- `HomeBG` - 首页背景图组件

## 🏗️ 项目架构

### 文件结构

```bash
src/
├── assets/       # 产品数据和类型定义
├── components/   # 可复用的 UI 组件
├── pages/        # 页面级组件，对应路由
├── stores/       # Zustand 状态管理
├── utils/        # 工具函数
└── App.tsx       # 主应用组件
```

### 状态管理

- 用户会话：姓名录入与本地存储
- 购物车：添加/删除商品、数量管理
- 订单追踪：配送状态更新、时间模拟
- 数据持久化：本地存储同步状态

### 数据模型

- `Product`：包含名称、辣度、图片、描述、价格等字段
- `CartItem`：购物车内每一项商品的数量、关联商品 ID
- `Order`：订单状态、用户信息、时间戳等
- `CustomerInfo`：姓名、电话、地址等信息

- 响应式布局：手动调整布局，适配桌面与移动端

### 设计系统

#### 字体排版

- 字体：Inter (Google Fonts)
- 字重：400 / 500 / 600 / 700

#### 配色方案

- 主色调：红色 `#ef4444`, `#dc2626`
- 背景色：黑色 `#000000`、灰色 `#f5f5f5`

#### 响应式断点

- 移动端：默认样式（< 1024px）
- 桌面端：`lg:` 前缀（≥ 1024px）

#### 自定义工具类

- 自定义尺寸：如 `h-88`, `w-116`（rem-based）
- 描边阴影：`text-shadow`, `box-shadow`

## 🔧 自定义配置

### 添加新商品

1. 修改 `src/assets/products.ts` 和 `productsWithPrice.ts`
2. 图片添加至 `public/` 目录
3. 在对应商品对象中添加 `spiceLevel` 和 `fav` 标志

### 样式修改

- 调整颜色：直接修改 Tailwind 类名（如 `bg-red-600`）
- 修改字体：修改组件类的 `font-bold`, `text-lg` 等
- 修改间距：调整 `px-`, `py-`, `gap-` 等类

### ✅ 环境要求

- Node.js 20+
- npm 或 yarn

---

### 📦 安装步骤

1. **克隆仓库**

   ```bash
   git clone <repository-url>
   cd bbq-chicken
   ```

2. **安装依赖**

   ```bash
   npm install
   ```

3. **启动开发服务器**

   ```bash
   npm run dev
   ```

4. **构建生产版本**

   ```bash
   npm run build
   ```

5. **预览生产版本**

   ```bash
   npm run preview
   ```
